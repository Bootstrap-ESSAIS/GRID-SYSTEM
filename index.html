   Lorsque le total des 12ème sur une row, fait moins de 12, il reste des blancs. Dans ce cas, on peut choisir d'ajouter un offset de tant d'unités avant une colonne (depuis le class=.. de celle-ci), pour la déplacer d'autant vers la droite. La class CSS à lui ajouter est : offset-N , avec N le nb. d'unités (12ème) de décalage. Enfin, tout comme pour les col-... il existe aussi des noms de classes du genre par exemple offset-md-N, signifiant ici par exemple que, l'offset ne prendra effet que si la largeur du ViewPort est >= la taille d"un ViewPort Medium.

col-sm-2 précédé d'un offset de 1 unité (offset-md-1).
col-sm-5 précédé d'un offset de 2 unités (offset-md-2).
